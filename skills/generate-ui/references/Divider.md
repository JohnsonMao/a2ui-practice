# Divider

## Props

| Prop | Type | Description | Default | Required |
|------|------|-------------|---------|----------|
| `axis` | "horizontal" \| "vertical" | Orientation of the divider line. Use vertical between side-by-side elements. | — | — |

## JSON Example

```json
{
  "id": "my-divider",
  "component": "Divider"
}
```

**Optional props:** `axis`
