# Card

## Props

| Prop | Type | Description | Default | Required |
|------|------|-------------|---------|----------|
| `child` | string | ID of the child component rendered inside the card body. | — | ✓ |

## JSON Example

```json
{
  "id": "my-card",
  "component": "Card",
  "child": "<component-id>"
}
```
