# Slider

## Props

| Prop | Type | Description | Default | Required |
|------|------|-------------|---------|----------|
| `label` | string \| expression | Optional label shown above the slider. | тАФ | тАФ |
| `min` | number | Minimum slider value. | тАФ | тАФ |
| `max` | number | Maximum slider value. | тАФ | тЬУ |
| `value` | number \| expression | Current slider value. | тАФ | тЬУ |

## JSON Example

```json
{
  "id": "my-slider",
  "component": "Slider",
  "max": 0,
  "value": 50
}
```

**Optional props:** `label`, `min`
