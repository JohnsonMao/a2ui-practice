# CheckBox

## Props

| Prop | Type | Description | Default | Required |
|------|------|-------------|---------|----------|
| `label` | string \| expression | Optional text label displayed next to the checkbox. | — | — |
| `value` | boolean \| expression | Current checked state of the checkbox. | — | ✓ |

## JSON Example

```json
{
  "id": "my-checkbox",
  "component": "CheckBox",
  "value": false
}
```

**Optional props:** `label`
