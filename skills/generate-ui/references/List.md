# List

## Props

| Prop | Type | Description | Default | Required |
|------|------|-------------|---------|----------|
| `direction` | "vertical" \| "horizontal" | Layout direction for list items. | — | — |
| `children` | array \| object | Ordered child component IDs rendered as list items. | — | — |

## JSON Example

```json
{
  "id": "my-list",
  "component": "List"
}
```

**Optional props:** `direction`, `children`
