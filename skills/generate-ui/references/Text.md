# Text

## Props

| Prop | Type | Description | Default | Required |
|------|------|-------------|---------|----------|
| `text` | string \| expression | Text content to render. | — | ✓ |
| `variant` | "h1" \| "h2" \| "h3" \| "h4" \| "h5" \| "caption" \| "body" | Typography style variant used to render the text. | — | — |

## JSON Example

```json
{
  "id": "my-text",
  "component": "Text",
  "text": "Sample text"
}
```

**Optional props:** `variant`
