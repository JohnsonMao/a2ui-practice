# Icon

## Props

| Prop | Type | Description | Default | Required |
|------|------|-------------|---------|----------|
| `name` | string \| expression | Icon identifier. Use values like "add", "search", "settings", or "warning". | — | ✓ |

## JSON Example

```json
{
  "id": "my-icon",
  "component": "Icon",
  "name": "star"
}
```
