# Tabs

## Props

| Prop | Type | Description | Default | Required |
|------|------|-------------|---------|----------|
| `tabs` | array | Ordered list of tabs and their content component IDs. | — | ✓ |

## JSON Example

```json
{
  "id": "my-tabs",
  "component": "Tabs",
  "tabs": [
    {
      "title": "Tab 1",
      "child": "<content-id>"
    }
  ]
}
```
