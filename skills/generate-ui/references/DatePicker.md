# DatePicker

## Props

| Prop | Type | Description | Default | Required |
|------|------|-------------|---------|----------|
| `date` | string \| expression | Currently selected date as an ISO 8601 date string (e.g. "2024-01-15"). Leave unset or empty for no initial selection. | — | — |
| `placeholder` | string | Placeholder text shown when no date is selected. Defaults to "Pick a date". | — | — |

## JSON Example

```json
{
  "id": "my-datepicker",
  "component": "DatePicker"
}
```

**Optional props:** `date`, `placeholder`
